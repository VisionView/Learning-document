# PHP(Personal Home Page)

## 一、PHP介绍

### 1、PHP 功能介绍

- 开发周期短，效率高，入门简单
- web服务器的开发语言，用来实现用户的请求
- 开源软件，所有操作系统稳定执行
- 实现面向过程(上学)，面向对象(洗车的过程)
- 支持的主流数据库：MySQL、Oracle等

### 2、开发环境

#### 1）WAMP

​		Windows + Apache + MySQL + PHP

#### 2）LAMP

​		Linux + Apache + MySQL + PHP
